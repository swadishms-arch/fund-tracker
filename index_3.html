Map each fund to its AMFI scheme code, then paste transactions: 2023-04-15, 10000 (date, amount ₹).

Overlap. Overlap % = Σ min(weight in A, weight in B) across shared stocks — the slice of two portfolios that's identical exposure. Names are normalized for matching (Ltd = Limited); near-matches are flagged, never auto-merged.

Returns. Live NAVs come from the free api.mfapi.in service, called straight from your browser. Units are derived from each transaction's amount and the NAV on that date; XIRR annualizes the whole cashflow stream. If your network blocks the API, set the current NAV manually on any fund and it still computes. NAV/units are estimates for tracking — your CAS statement is the record of truth. Importing a Detailed CAS reads every transaction directly; dividend reinvestments are counted as buys, which can slightly overstate “invested”.

Private & free. Nothing leaves your device except the NAV lookups. Auto-saves in this browser; use Export for a backup. Host free: drop this index.html into a GitHub repo → Settings → Pages → deploy from the main branch.
